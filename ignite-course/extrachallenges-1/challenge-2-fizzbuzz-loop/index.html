Northcoders Ignite: Extra Challenges

2: Fizzbuzz with a loop

For this challenge you will need to define a function that takes 1 argument.

The argument should be the number you want to play fizzbuzz until.

You should use a loop to console log either fizz, buzz, fizzbuzz or the normal number for each number up until the maximum number passed into the function.
